    API Reference

    Base URL

    http://localhost:8000

    Authentication

    No authentication required (runs locally). For production, add your own auth middleware.

    Endpoints

    Health Check

    GET /health

    Response: 

    {
  "status": "ok",
  "service": "quant-engine"
}

    Start Research Run

    POST /research/start
Content-Type: application/json

    Request Body: 

    {
  "config": {
    "name": "My Research Run",
    "trading_styles": ["intraday", "swing"],
    "data": {
      "openalgo": {
        "host": "http://127.0.0.1:5000",
        "api_key": "your-key-here",
        "source": "db"
      },
      "symbols": [
        {"symbol": "NIFTY", "exchange": "NSE"}
      ],
      "timeframes": ["15m", "1h"],
      "start_date": "2022-01-01",
      "end_date": "2024-12-31"
    },
    "generation": {
      "target_count": 10000,
      "max_conditions_per_entry": 3
    },
    "filters": {
      "backtest": {
        "min_trades": 30,
        "min_sharpe": 1.0,
        "max_drawdown": 0.25
      }
    },
    "ranking": {
      "mode": "robustness_first"
    }
  },
  "run_id": "my_custom_id"
}

    Response: 

    {
  "run_id": "my_custom_id",
  "status": "started",
  "progress": {}
}

    Stop Research Run

    POST /research/stop/{run_id}

    Response: 

    {
  "status": "stopping",
  "run_id": "run_abc123"
}

    Export Strategy

    POST /research/export/{run_id}/{strategy_id}

    Response: 

    {
  "strategy_id": "abc12345",
  "message": "Strategy exported",
  "export_dir": "./data/runs/run_abc123/exports"
}

    List All Runs

    GET /research/runs

    Response: 

    {
  "runs": ["run_abc123", "run_def456", "run_ghi789"]
}

    Error Responses

    All errors follow this format:

    {
  "detail": "Run run_xyz not found"
}

    Status Codes: | Code | Meaning | |------|---------| | 200 | Success | | 404 | Run or strategy not found | | 422 | Validation error (invalid config) | | 500 | Internal server error |
